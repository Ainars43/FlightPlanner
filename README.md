# Flight Planner
 
 FlightPlanner is a simple planner using web api which can be used by flight companies.
 Application is still in progress.
 
 ## About the project
 
 This is a first iteration of web api application which was created as part of the CODELEX intensive course C#/.Net.
 Next will be adding the application to the frontend using AngularJS.
 
 This app contains two web api's: Admin Api and Customer Api.
